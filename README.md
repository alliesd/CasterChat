# CasterChat
Connects CasterLabs Chat and minecraft Via Fabric

Add mod to minecraft mods folder

minecraft_version=26.1.2
Fabric loader_version=0.19.3

1. Start Caffeinated
2. go to chat bot
3. Add command - any platform - send a message execute and input caster code
4. Double-click start_bridge.bat
5. Launch Minecraft
6. Stream chat appears in-game
